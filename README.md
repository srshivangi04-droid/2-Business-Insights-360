# Business Insights 360

This project analyzes overall business performance using Power BI.

## KPIs Used:
- Total Revenue
- Total Profit
- Profit Margin %
- Sales by Region
- Category Performance
- Customer Segment Analysis

## Key Insights:
- West region contributes highest revenue.
- Technology category has highest profit margin.
- Corporate segment generates stable recurring revenue.
- Profit margin drops during discount-heavy months.
